# event-and-media-management-paltform1
A centralized platform for clubs and societies to upload, organize, search, and manage event photos and videos
<br>
## Features
- User Registration and Login email otp verification
- JWT Authentication
- Role-based Users: Admin, Photographer, Club Member, Viewer
- Event Creation and Management
- Media Upload System
- Cloudinary Cloud Storage
- Media Gallery
- AI-based Image Tags
- Search by Tags
- Like Button with Counter
- Comment System
- Add to Favorites
- Download Media
- Notifications
- Dashboard Statistics
- facial recognization
- MongoDB Database Integration
## Tech Stack
### Frontend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Multer
- Cloudinary
- Sharp
